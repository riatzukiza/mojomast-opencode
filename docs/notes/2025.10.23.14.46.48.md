Based on my analysis of the environment variables in this OpenCode project, here are the main categories of
environment variables available:

## Core OpenCode Configuration

• OPENCODE_CONFIG - Custom config file path
• OPENCODE_CONFIG_CONTENT - JSON config content as string
• OPENCODE_PERMISSION - Permission settings JSON
• OPENCODE_CALLER - IDE caller identification (e.g., "vscode")
• OPENCODE_BIN_PATH - Binary path for shell wrappers
• OPENCODE_BASE_URL - API base URL for SDK clients

## Feature Flags

• OPENCODE_AUTO_SHARE - Enable automatic sharing
• OPENCODE_DISABLE_AUTOUPDATE - Disable auto-updates
• OPENCODE_DISABLE_PRUNE - Disable message pruning
• OPENCODE_DISABLE_DEFAULT_PLUGINS - Disable default plugins
• OPENCODE_DISABLE_LSP_DOWNLOAD - Disable LSP server downloads
• OPENCODE_ENABLE_EXPERIMENTAL_MODELS - Enable experimental models
• OPENCODE_DISABLE_AUTOCOMPACT - Disable auto-compaction
• OPENCODE_EXPERIMENTAL_WATCHER - Enable experimental file watcher
• OPENCODE_EXPERIMENTAL_TURN_SUMMARY - Enable turn summaries
• OPENCODE_EXPERIMENTAL_NO_BOOTSTRAP - Disable project bootstrapping

## Development & Build

• OPENCODE_VERSION - Version string
• OPENCODE_CHANNEL - Release channel
• OPENCODE_TUI_PATH - Path to TUI binary
• OPENCODE_INSTALL_DIR - Custom installation directory
• OPENCODE_SERVER - Server URL for TUI
• OPENCODE_API - API endpoint
• OPENCODE_AGENTS_SWITCH_SINGLE_MODEL - Force single model for agents
• OPENCODE_THEME - TUI theme setting

## Frontend (Vite)

• VITE_API_URL - API URL for web frontend
• VITE_AUTH_URL - Authentication URL
• VITE_OPENCODE_SERVER_HOST - Desktop app server host
• VITE_OPENCODE_SERVER_PORT - Desktop app server port

## Provider API Keys

• ANTHROPIC_API_KEY - Anthropic Claude API key
• AWS_ACCESS_KEY_ID - AWS access key
• AWS_SECRET_ACCESS_KEY - AWS secret key
• AWS_PROFILE - AWS profile name
• AWS_BEARER_TOKEN_BEDROCK - AWS Bedrock bearer token
• AWS_REGION - AWS region
• GOOGLE_APPLICATION_CREDENTIALS - Google Cloud service account credentials
• GOOGLE_CLOUD_PROJECT - Google Cloud project ID
• GOOGLE_CLOUD_LOCATION - Google Cloud region
• VERTEX_LOCATION - Vertex AI location alias

## Infrastructure & Services

• SLACK_BOT_TOKEN - Slack bot token
• SLACK_SIGNING_SECRET - Slack signing secret
• SLACK_APP_TOKEN - Slack app token
• STRIPE_SECRET_KEY - Stripe payment processing
• POSTHOG_KEY - Analytics tracking

## GitHub Actions

• GITHUB_TOKEN - GitHub API token
• GITHUB_RUN_ID - Workflow run identifier
• MODEL - Model selection
• SHARE - Share configuration

## VS Code Extension

• _EXTENSION_OPENCODE_PORT - Extension port communication

The environment variables follow standard naming conventions and support comprehensive configuration of the
OpenCode ecosystem across development, deployment, and runtime scenarios.
