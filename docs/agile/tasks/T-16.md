## Description

## Requirements
- [ ] When `OPENCODE_PERF_TIMING=1`, include `ts.gen` at publish and `ts.sse` at write; benches compute `latencyMs = ts.sse - ts.gen`.
- [ ] Ensure no production overhead when flag is off.

## DoD
